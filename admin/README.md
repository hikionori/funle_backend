  - Редатирование
  - Удаление
  - Просмотр

* Работа с лекциями
  - Создание
  - Редатирование
  - Удаление
  - Просмотр

* Работа с курсами
  - Создание
  - Редатирование
  - Удаление
  - Просмотр
